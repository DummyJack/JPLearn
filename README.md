# 日語學一學(開發中)

## 安裝套件

```bash
pip install -r requirements.txt
```

## 啟動

```
執行 run.py
```

## 測試

### 安裝測試套件

```bash
pip install -r tests/requirements-dev.txt
```

### 執行測試

```bash
python -m pytest
``` 

### 生成覆蓋率報告

```bash
python -m pytest --cov=functions --cov-report=term-missing --cov-report=html
```
